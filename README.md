# Intro to Digital Forensics – TryHackMe Writeup

This repository contains my writeup for the TryHackMe room **Intro to Digital Forensics**.

The room introduces core digital forensics concepts and demonstrates how metadata can support an investigation.

## Topics Covered

* Digital evidence and forensic relevance
* Chain of custody
* PDF metadata analysis using pdfinfo
* Image metadata analysis using exiftool
* GPS coordinates extracted from EXIF data
* Identifying a camera model from image metadata
